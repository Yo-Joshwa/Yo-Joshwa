### 📂 Projects

#### 🧳 [PathCrafters](https://github.com/Yo-Joshwa/PathCrafters) — *Oct 2025 – Apr 2026*
An AI-powered travel planner to discover destinations, generate itineraries, and manage trips.

**Tech Stack:** React.js, Node.js, Express.js, Tailwind CSS, MongoDB, Gemini API, Clerk

- Built a full-stack travel planning web app with AI-driven itinerary generation using the Gemini 2.5 Flash API, enabling users to create personalized day-wise travel schedules based on destination, budget, travel style, and interests, with structured JSON parsing and error handling on the backend.
- Designed and implemented a RESTful API with Express.js covering trip CRUD operations, user authentication via Clerk (JWT middleware), city/attraction discovery, and a weather-integrated climate scoring system with a custom scoring algorithm that ranked months by temperature, rainfall, and crowd metrics.
- Built a climate caching pipeline with a MongoDB-backed Destination Climate model and a node-cron job that auto-refreshes stale climate data weekly, reducing redundant third-party API calls and improving response performance for destination recommendations.
- Developed an AI-powered smart packing list generator that dynamically infers season from trip start date, extracts unique activity types from saved itineraries, and prompts Gemini to return category-wise, destination-specific packing suggestions with essentials flagging — all consumed via a responsive React frontend.
